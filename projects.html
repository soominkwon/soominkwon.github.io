Projects

Below are some of the course projects that I have been a part of during my time at Rutgers University.

Course Projects

  • Spectral Initialization for Phase Retrieval: Theory and Proofs
    - Soo Min Kwon, Nitya Sathyavageeswaran
    - Course: High-Dimensional Probability

    - Using concentration inequalities to prove the initialization guarantees of AltMinPhase.


  • A Friendly Introduction to Differential Privacy
    - Soo Min Kwon
    - Course: Detection and Estimation Theory

    - "Gentle" introduction and explanation to some of the main topics in differential privacy.


  • On the Robustness of Multi-view Convolutional Neural Networks with Applications to Image Classification
    - Soo Min Kwon, Ye Tao, Yu Wu
    - Course: Machine Vision

    - Evaluated the performance of various architectures of multi-view convolutional neural networks (CNN) with different image datasets.


  • Nuclear Norm and Alternating Minimization for Matrix Completion
    - Soo Min Kwon
    - Course: Convex Optimization

    - Studied the matrix completion problem and two different optimization methods for solving this problem.


  • Classification and Clustering
    - Soo Min Kwon, Brian Cheng, Manish Kewalramani
    - Course: Machine Learning for Engineers

    - Used different machine learning algorithms for classification and clustering on several datasets.


  • Mobile Edge Computing Using Raspberry Pi for Small-Scale Testbed
    - Soo Min Kwon, Natalie Kim, Manish Kewalramani
    - Course: Wireless Communications

    - Explored the benefits of machine learning applications on mobile edge computing using a Raspberry Pi.
